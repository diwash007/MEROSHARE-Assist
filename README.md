# MEROSHARE Assist

helps to quickly login and apply IPO on Meroshare

## Prerequisites

- [node + npm](https://nodejs.org/)

## Setup

```
npm install
```

## Build

```
npm run build
```

## Build in watch mode

### terminal

```
npm run watch
```

### Visual Studio Code

Run watch mode.

type `Ctrl + Shift + B`

## Load extension to chrome

Load `dist` directory

## Test

`npx jest` or `npm run test`
